# Coordiante-System


## Download Project & Install Packages


```sh
$ git clone https://github.com/deepaksaini14896/test.git

$ pip install -r requirements.txt
```

```sh
$ cd interview_1

$ uvicorn sql_app.main:app --reload

$ pytest
```
